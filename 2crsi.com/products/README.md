# 2crsi.com Scraper

Install dependencies:
pip install -r requirements.txt

Run product page:
python crawl.py --url https://2crsi.com/catalog/mona-1-12-gg-57421?category=17 --out output/part

Run category page:
python crawl.py --url https://2crsi.com/catalog/category/rackmount-servers-15 --out output/category
