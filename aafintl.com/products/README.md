# AAF International Scraper

## Setup
```bash
pip install -r requirements.txt
```

## Usage
```bash
# Category page
python crawl.py --url "https://www.aafintl.com/us/products/energy-and-industrial-air-quality" --out output/category

# Product page  
python crawl.py --url "https://www.aafintl.com/us/products/energy-and-industrial-air-quality/diesel-and-gas-engine-systems/multiduty--oil-bath-filter" --out output/part
```
