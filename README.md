# Web Data Extraction Tool

Python crawler to extract structured data from webpages.

## Installation

pip install -r requirements.txt

## Usage

Category page:

python3 crawl.py --url https://americanbeautytools.com/Soldering-Irons --out output/category

Product page:

python3 crawl.py --url https://americanbeautytools.com/soldering-iron-3108/1/features --out output/part
